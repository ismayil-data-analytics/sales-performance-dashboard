# Sales Performance Dashboard

## Project Overview

This project is an interactive Power BI dashboard designed to analyze sales performance, profitability, customer behavior, and product insights.

## Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling

## Dashboard Pages

### 1. Executive Summary

- Overview of key sales metrics
- KPI cards for Sales, Profit, Quantity, and Orders
- Sales trend analysis
- Category and product performance overview
- Interactive slicers and filters

### 2. Customer & Product Analysis

- Customer performance analysis
- Product-level analysis
- Top-performing products
- Detailed customer and product insights
- Interactive filtering

## Key Features

- Data cleaning and transformation using Power Query
- Data modeling and relationships
- Custom DAX measures
- Interactive slicers
- Bookmark functionality for resetting filters
- Page navigation between dashboard sections
